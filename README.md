# Shopify Icon Snippets

Add icon snippets from [Lucide](https://lucide.dev), [Heroicons](https://heroicons.com), and more to your Shopify theme.

## Packages

| Package                        | Description        |
| ------------------------------ | ------------------ |
| [@ckreidl/sis](./packages/cli) | CLI tool           |
| [docs](./packages/docs)        | Documentation site |

## Quick Start

```bash
npx @ckreidl/sis add lucide menu chevron-down
```

See the [CLI README](./packages/cli/README.md) for full usage.

## Development

```bash
bun install
```

### CLI

```bash
cd packages/cli
bun run dev
```

### Docs

```bash
cd packages/docs
bun run dev
```
