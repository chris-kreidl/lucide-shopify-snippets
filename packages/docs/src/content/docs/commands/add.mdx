The `add` command generates Liquid snippet files for the specified icons.

## Usage

<Code lang="bash" code="npx @ckreidl/sis add <library>[:<variant>] <icons...> [options]" />

## Arguments

| Argument  | Description                                                |
| --------- | ---------------------------------------------------------- |
| `library` | Desired icon library                                       |
| `variant` | Optional. Desired icon variant, if library offers variants |
| `icons`   | One or more icon names (e.g., `menu`, `chevron-down`)      |

## Options

| Option                  | Description              | Default    |
| ----------------------- | ------------------------ | ---------- |
| `-d, --dir <path>`      | Output directory         | `snippets` |
| `-p, --prefix <prefix>` | Filename prefix          | `icon-`    |
| `-f, --force`           | Overwrite existing files | `false`    |

## Examples

Add a single icon:

<Code lang="bash" code="npx @ckreidl/sis add lucide menu" />

Add multiple icons:

<Code lang="bash" code="npx @ckreidl/sis add lucide menu chevron-down arrow-right" />

Custom output directory:

<Code lang="bash" code="npx @ckreidl/sis add lucide menu --dir theme/snippets" />

Specify variant:

<Code lang="bash" code="npx @ckreidl/sis add heroicons:16/solid arrow-down" />

Custom prefix:

<Code lang="bash" code="npx @ckreidl/sis add lucide menu --prefix lucide-
# Creates: lucide-menu.liquid" />
