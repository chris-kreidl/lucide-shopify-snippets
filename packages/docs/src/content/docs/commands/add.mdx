The `add` command generates Liquid snippet files for the specified icons.

## Usage

<Code lang="bash" code="npx @ckreidl/lucide-shopify-snippets add <icons...> [options]" />

## Arguments

| Argument | Description |
|----------|-------------|
| `icons` | One or more icon names (e.g., `menu`, `chevron-down`) |

## Options

| Option | Description | Default |
|--------|-------------|---------|
| `-d, --dir <path>` | Output directory | `snippets` |
| `-p, --prefix <prefix>` | Filename prefix | `icon-` |
| `-f, --force` | Overwrite existing files | `false` |

## Examples

Add a single icon:

<Code lang="bash" code="npx @ckreidl/lucide-shopify-snippets add menu" />

Add multiple icons:

<Code lang="bash" code="npx @ckreidl/lucide-shopify-snippets add menu chevron-down arrow-right" />

Custom output directory:

<Code lang="bash" code="npx @ckreidl/lucide-shopify-snippets add menu --dir theme/snippets" />

Custom prefix:

<Code lang="bash" code="npx @ckreidl/lucide-shopify-snippets add menu --prefix lucide-
# Creates: lucide-menu.liquid" />
