The `search` command helps you find icons by name or tag.

## Usage

<Code lang="bash" code="npx @ckreidl/sis search <library> <term> [options]" />

## Arguments

| Argument  | Description       |
| --------- | ----------------- |
| `library` | Library of choice |
| `term`    | Search term       |

## Options

| Option      | Description                       |
| ----------- | --------------------------------- |
| `-t, --tag` | Search tags instead of icon names |

## Examples

### Search for an icon:

<Code lang="bash" code="npx @ckreidl/sis search lucide arrow" />

Output:

<Code lang="plaintext" code="Found similar: arrow-up, arrow-down, arrow-left, arrow-right, arrow-up-left" />

The search uses fuzzy matching, so partial or misspelled names will still return relevant results.

### Search tags:

<Code lang="bash" code="npx @ckreidl/sis search lucide -t text" />

Output:
