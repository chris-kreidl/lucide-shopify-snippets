The `search` command helps you find icons by name.

## Usage

<Code lang="bash" code="npx @ckreidl/lucide-shopify-snippets search <icon>" />

## Arguments

| Argument | Description             |
| -------- | ----------------------- |
| `icon`   | Icon name to search for |

## Examples

Search for an icon:

<Code lang="bash" code="npx @ckreidl/lucide-shopify-snippets search arrow" />

Output:

<Code lang="plaintext" code="Found similar: arrow-up, arrow-down, arrow-left, arrow-right, arrow-up-left" />

The search uses fuzzy matching, so partial or misspelled names will still return relevant results.
