The `tags` command lists all available tags from the Lucide icon library, along with the number of icons associated with each tag.

## Usage

<Code lang="bash" code="npx @ckreidl/sis <library> tags" />

## Arguments

| Argument  | Description       |
| --------- | ----------------- |
| `library` | Library of choice |

## Output (truncated):

<Code lang="plaintext" code={`Found the following tags:
    * accessibility [4 icons]
    * account [17 icons]
    * action [5 icons]
    * add [22 icons]
    * advertising [1 icon]
    * agriculture [7 icons]
    ...`} />

## Use Cases

- Discover what categories of icons are available
- Find the right tag to use with the `search --tag` command
- Explore the Lucide icon library by category
