The `tags` command lists all available tags from the selected icon library, along with the number of icons associated with each tag.

## Usage

<Code lang="bash" code="npx @ckreidl/sis tags <library>" />

## Arguments

| Argument  | Description       |
| --------- | ----------------- |
| `library` | Library of choice |

## Output (truncated):

Example output for the Lucide library:

<Code lang="plaintext" code={`Found the following tags:
    * accessibility [4 icons]
    * account [17 icons]
    * action [5 icons]
    * add [22 icons]
    * advertising [1 icon]
    * agriculture [7 icons]
    ...`} />

## Use Cases

- Discover what categories of icons are available
- Find the right tag to use with the `search --tag` command
- Explore icon libraries by category
