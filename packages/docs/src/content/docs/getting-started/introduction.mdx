**Lucide Shopify Snippets** is a CLI tool that generates Shopify Liquid snippets for various icon libraries.

## Why?

Shopify themes don't have a built-in way to use icon libraries. This tool generates Liquid snippets that render icons as inline SVGs, with support for:

- **Customizable size** — Set width and height via Liquid parameters
- **CSS classes** — Add Tailwind or custom classes
- **Stroke width** — Adjust line thickness
- **Icon Libraries** - Currently supports Lucide, Heroicons

## Quick Start

<Code lang="bash" code="npx @ckreidl/sis lucide add menu chevron-down" />

This creates snippets using Lucide icons in your `snippets/` directory:

<FileTree>
- snippets
  - icon-menu.liquid
  - icon-chevron-down.liquid
</FileTree>

Use them in your theme:

<Code lang="liquid" code="{% render 'icon-menu' %}
{% render 'icon-menu', size: 32, class: 'text-primary' %}" />
