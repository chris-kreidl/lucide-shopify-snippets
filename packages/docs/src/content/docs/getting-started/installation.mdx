## Using npx (Recommended)

No installation required. Run directly in your Shopify theme directory:

<Code lang="bash" code="npx @ckreidl/lucide-shopify-snippets add menu" />

## Global Installation

<Steps>
1. Install globally to use without `npx`:

   <Code code="npm install -g @ckreidl/lucide-shopify-snippets" lang="bash" />

2. Then run:

      <Code code="lucide-shopify-snippets add menu" lang="bash" />
</Steps>

## Requirements

- Node.js 18 or later
- A Shopify theme directory with a `snippets/` folder
