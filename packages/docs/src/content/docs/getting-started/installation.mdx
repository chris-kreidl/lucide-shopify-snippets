## Using npx (Recommended)

No installation required. Run directly in your Shopify theme directory:

<Code lang="bash" code="npx @ckreidl/sis lucide add menu" />

## Global Installation

<Steps>
1. Install globally to use without `npx`:

      <Code code="npm install -g @ckreidl/sis" lang="bash" />

2.  Then run:

      <Code code="sis lucide add menu" lang="bash" />

</Steps>

## Requirements

- Node.js 18 or later
- A Shopify theme directory with a `snippets/` folder
